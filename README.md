# Mahipal Puri — Portfolio

Personal portfolio website built with plain HTML, CSS and JavaScript, with 3D enhancements via Three.js.

**Live:** _(your Vercel URL after deploy)_

## Highlights

- Modern dark + glassmorphism UI with light-mode toggle
- Three.js full-page particle background with cursor parallax
- Three.js animated wireframe scene (torus knot + icosahedron + ring) in the hero
- 3D mouse-tilt + glare on project cards
- Animated typewriter, scroll-reveal, animated skill bars
- 15 GitHub projects with live Vercel demos
- Working contact form via Web3Forms
- Mobile responsive + reduced-motion friendly

## Tech

HTML5 · CSS3 (custom properties, glassmorphism) · Vanilla JS · Three.js (CDN) · Web3Forms

## Local preview

Open `index.html` directly in a browser — no build step.
